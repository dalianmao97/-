mac 搭建hexo 新手学习

Veröffentlicht am 2018-01-31

安装自制软件homebrew

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

安装nodejs

brew install node

安装hexo

sudo npm install -g hexo

密码为电脑admin密码

创建博客文件夹

  • 1 mkdir blog
  • 2 cd blog

  • 3 hexo init

    此时blog文件下会生成很多文件和文件夹，如下图

生成一套静态网页

hexo generate

在服务器上运行

hexo server

这时就可以在浏览器中输入http://localhost:4000查看网站首页

配置

与github账户关联
首先你得有一个github账号,并新建一个名为[your_arrount].github.io的仓库。
然后打开前面创建的blog目录下的_config.yml文件，再修改最下方的deploy为：

冒号后面一定要给空格

* deploy:
   type: git
   repo: https://github.com/your_arrount/your_arrount.github.io.git
   branch: master

安装hexo的git部署

npm install hexo-deployer-git -S

最后将生成静态页面部署到github的仓库中

hexo d -g

当提示info deploy done:git 即上传成功，这时就可以通过https://your_account.github.io来
访问你的博客了

  • 1 hexo clean
