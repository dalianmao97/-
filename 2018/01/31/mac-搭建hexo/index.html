###安装nodejs
brew install node

###安装hexo
sudo npm install -g hexo

密码为电脑admin密码


###创建博客文件夹


  • 1 mkdir blog

  • 2 cd blog

  • 3 hexo init

###生成一套静态网页
hexo generate



在服务器上运行



hexo server


###这时就可以在浏览器中输入http://localhost:4000查看网站首页
